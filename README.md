## Description

პროექტს ჭირდება mysql-ის ბაზა
ასევე უნდა შეიქმნას .env ფაილი რომლის ველების კოპირება შეგიძლიათ .env.example ფაილიდან example ფაილი სატესტოდ შევსებულია

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
